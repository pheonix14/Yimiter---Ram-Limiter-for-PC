# Yimiter

Yimiter is an open-source RAM limiter application for desktop PCs.

## Author
Created by **pheonix14**

## Purpose
This project is created for **help and educational purposes** only. 

## Privacy Policy
**No data is collected.** We respect your privacy. This application operates entirely locally on your machine and does not send any telemetry, usage data, or personal information over the internet.
